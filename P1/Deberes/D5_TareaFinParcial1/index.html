Valores

Definición

Son datos que almacenamos en variables.

El tipo de dato es una propiedad de un valor que determina valores que puede tomar, que tipos de operaciones podemos realizar sobre este y cómo su representación interna.

Tipos de datos

JavaScript es un lenguaje de tipado débil o dinámico. Esto significa que no es necesario declarar el tipo de variable antes de usarla. En los lenguajes de tipado dinámico como JavaScript o Phyton la comprobación de los tipos se realiza durante la ejecución del programa en vez de durante la compilación.

Sintaxis

  • String: "Hello world" 'Hello world' `Hello ${name}`

  • Number: 10

  • Boolean: false

  • Null: null

  • Undefined: undefined

  • Array: ["María", "José", "Carlos"]

  • Objeto: 

    {
  nombre: "Marcel",
  edad: "38",
  ciudad: "Alaior",
}

Ejemplos

console.log(typeof "Carlos"); //'string'
console.log(typeof 34); //'number'
console.log(typeof false); //'boolean'
console.log(typeof null); //'object'
console.log(typeof undefined); //'undefined'
console.log(typeof ["María", "José", "Carlos"]); //'object'
console.log(typeof {name: "Juan", age: 18}); //'object'
